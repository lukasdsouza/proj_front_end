2-Soluçao do problema: TEXT HAS MINIMUM CONTRAST

HTML Pol. de Privacidade CSS .text-green { color: #00FF00; / Cor verde / }

.bg-white { background-color: #FFFFFF; / Cor branca / }

classe bg-white para definir o plano de fundo como branco e a classe text-green para definir a cor do texto como verde.

Compartilhar

classe text-secondary-color para definir a cor do texto como cinza escuro, classe text-xs define o tamanho do texto como extra pequeno e font-bold define o texto como negrito.

ou com seu e-mail.

classe text-slate-400 para text-slate-900 para aumentar o contraste e tornar o texto mais escuro, tamanho do texto para text-lg para torná-lo mais legível e classe font-bold foi substituída por font-semibold para manter a negrito, mas com uma intensidade menor, para uma melhor legibilidade.
